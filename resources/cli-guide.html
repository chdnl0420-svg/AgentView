Claude Code CLI 설치 가이드

AgentView 가 백그라운드 에이전트를 띄우려면 시스템에 claude CLI 가 설치돼 있어야 합니다. 인스톨러가 자동 설치를 시도했지만 Node.js / npm 이 시스템에 없으면 건너뜁니다.

1단계 · Node.js 설치

두 가지 방법 중 하나를 고르세요.

  • 공식 설치 프로그램 — nodejs.org 에서 LTS 버전 (18 이상). 설치 중 "Add to PATH" 옵션을 켜야 합니다.
  • nvm-windows — github.com/coreybutler/nvm-windows. 여러 Node 버전 전환이 편합니다.

설치 후 새 PowerShell 창을 열고 확인:

node -v
npm -v

2단계 · claude CLI 설치

관리자 권한 없이 사용자 영역에 전역 설치합니다.

npm install -g @anthropic-ai/claude-code

설치 후 버전 확인:

claude --version

OAuth 로그인:

claude login

3단계 · AgentView 재시작

AgentView 우상단의 상태 배지가 초록색 "Claude OK" 로 바뀌면 준비 완료입니다. 그렇지 않으면 AgentView 를 한 번 종료하고 다시 실행하세요.

문제 해결

  • npm 명령을 못 찾는다 — PowerShell 을 새 창으로 열어 보세요. 그래도 안 되면 Node 설치 시 "Add to PATH" 가 꺼졌을 가능성. 재설치하세요.
  • 회사 프록시 — npm config set proxy / https-proxy 로 프록시를 등록해야 합니다.
  • 인증서 오류 — 사내 SSL 인증서가 필요할 수 있습니다. IT 부서에 문의.

이 페이지는 AgentView 가 자동으로 띄웠습니다 — 닫아도 됩니다. 다시 보려면 옵션 패널의 "CLI 가이드 보기" 를 클릭하세요.
